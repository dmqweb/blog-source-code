# 图片上传预览

业务中经常有上传图片的需求在，有时我们需要去尽快预览要上传的图片，此时会有两种常见的方法：

# URL.createObjectURL

直接返回，同步执行，会为图片文件创建一个地址，存储在 document 中，并且不会被垃圾回收机制所回收。清除方式只有 unload 事件或 revokeObjectURL 进行手动清除。

# FileReader.readAsDataURL

通过回调的形式返回，异步执行，将图片文件转为 base64 格式，可以通过链接形式直接引用，会依据垃圾回收机制进行回收，
